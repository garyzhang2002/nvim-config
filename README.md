## Requirements

+ Neovim version **0.7.0** or higher.
+ A C compiler in your path.
+ Github CLI (authenticated) in your path.

## Usage

``` shell
git clone https://github.com/garyzhang2002/nvim-config.git ~/.config/nvim
```
